# BMW Group · Senior AI Strategy Lead (f/m/x) application

A tailored single-page site for the Senior AI Strategy Lead role at the BMW Group, by Vilol Joshi.

It answers each requirement from the posting with evidence, maps the six responsibilities to a concrete approach, and frames the work as AI strategy that turns breakthrough technology into road-ready, measurable impact. Dark, premium automotive theme with a car-on-road motion graphic, a trend-scanning radar, and M-tricolor accents.

## Run locally

```bash
python3 -m http.server 8911
# open http://localhost:8911
```

## Deploy (GitHub Pages)

Settings, then Pages, then Source: `main` / root. The site is plain static HTML, CSS, and JS.

## Files

| File | Purpose |
|------|---------|
| `index.html` | The application page |
| `styles.css` | Dark automotive theme, car/road/radar motion graphics |
| `main.js` | Scroll reveals, counters, cursor glow, rotator |
